# Reference models (локальные ассеты)

Эта папка предназначена для **локального** хранения “референс‑моделей” (изображения людей), которые вы используете для генераций и для воспроизводимых гайдов.

## Важно

- Содержимое `assets/reference_models/` **игнорируется git’ом** (см. `.gitignore`), чтобы случайно не запушить персональные изображения.
- Давайте файлам понятные имена и держите 1–2 “каноничных” варианта на каждого персонажа.

## Рекомендуемая структура имён

- `male_full_01.png`
- `female_portrait_01.jpg`
- `child_girl_full_01.png`
- `child_boy_full_01.png`

## Как использовать в скриптах/гайдах

- В гайдах ссылайтесь на эти файлы только в локальном контексте (для продакшен‑публикаций используйте отдельную папку с “публичными” ассетами).
- Скрипты из `skills/ui-guides-playwright/` умеют принимать пути к этим файлам через аргументы/переменные окружения (см. skill).
