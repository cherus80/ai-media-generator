# Шаблоны для гайдов и продающих статей (RU)

## 1) Гайд (пошаговая инструкция)

**Заголовок:** *[что сделает пользователь] за [время]*

**Вступление (2–4 предложения):**
- Для кого (новичок/профи)
- Что будет на выходе
- Что понадобится (фото / 30⭐ / браузер)

**Содержание:**
1. Войти в аккаунт
2. Открыть нужный раздел (примерка/редактирование)
3. Загрузить фото
4. Запустить генерацию
5. Скачать результат / поделиться

**Каждый шаг:**
- “Что нажать” (конкретные кнопки/поля)
- “Что увидеть” (ожидаемый результат)
- Скриншот (подпись + зачем он)

**Ошибки/решения (FAQ):**
- “Не входит в аккаунт” → проверить согласие ПДн/пароль
- “Не хватает звёзд” → пополнить/проверить баланс
- “Фото не загружается” → формат/размер/перезагрузка

**CTA:**
- 1 строка: “Откройте приложение и повторите шаги”
- Ссылка/QR (если нужно)

## 2) Продающая статья (маркетинг)

**Заголовок:** *[главная выгода] + [соц.доказательство/скорость]*

**Хук (первые 2 абзаца):**
- Боль/желание аудитории
- Обещание результата (без “магии”)

**Блок “Как это работает” (очень коротко):**
- 3 шага: загрузить → выбрать → получить
- 2–3 скриншота как “доказательство”

**Блок “Почему это удобно”:**
- Скорость
- Простота
- Качество
- Конфиденциальность/контроль (если актуально)

**Мини‑гайд внутри статьи (5–7 шагов):**
- Нумерованные шаги + скриншоты

**Цены/ограничения (если пользователь просил):**
- Коротко и прозрачно

**CTA:**
- “Попробуйте сейчас” + ссылка

## Правила качества

- Пиши простыми фразами, без “воды”.
- Скриншоты должны соответствовать шагам 1:1.
- Не показывай админку и не вставляй секреты/пароли.
